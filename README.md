Ola, esse repositorio trata-se de um projeto para as disciplinas Desenvolvimento BackEnd 1 e Desenvolvimento FrontEnd 1

Objetivo do projeto: O objetivo do projeto é botar em pratica tudo que foi aprendido nas materias citadas anteriormente, colocando em prova o meu conhecimento sobre as tecnologias aprendidas. 

Descricao: O projeto é um site  de avaliacoes e resenhas literarias, baseada na arquitetura MVC(Model View Controller)
    Model -> É onde a classes dos objetos sao criadas, com atributos, regras de negocio e Getters and Setters
    View  -> Sao as paginas que o usuario ira navegar no site
    Controller -> O controller recebera todos direcionamentos do index.php e decidira o que fara com as requicisoes
